### 注意点
1. 主线程执行完成之后，无论协程是否运行结束都会立刻停止运行
2. 查看是否有竞争
   ```bash
   go build -race 目标文件
   运行目标文件
   ```
   ![pic](pic/Snipaste_2020-10-26_14-56-07.png)
3. 当协程执行之后，主线程不知道协程已经执行完成了，因此有可能默认还存在竞争,所以在输出的时候还需要加锁
